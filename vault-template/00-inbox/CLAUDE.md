# Inbox

Drop anything here. Links, ideas, notes, screenshots.

**Goal:** Keep this empty by processing with `/digest`.

## How to Add Items

One file per item:
```
00-inbox/interesting-article.md
```

Contents:
```
https://example.com/article

Why I saved this: Quick note about what caught my attention
```

## Processing

Run `/digest` to:
1. Fetch content from URLs
2. Create structured note in `03-resources/`
3. Delete inbox file
